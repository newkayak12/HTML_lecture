html5에서 지원하는 type

            
1. type : Color

                    색상 선택하는 type
            
2. type : date, datetime, datetimeLocal, month, week, time

                    날짜를 입력받는 type
            
3. type : number
        
                    숫자 입력 전용
            
4. 기타 등등 타입 (search/range/url/email

                    -1. search : 검색 (검색어/ 텍스트 타입이랑 비슷한데 한 번에 지우는 버튼이 있음)
                    -2. range  : 범위 (일정 범위에 있는 값을 표시 max/min/step을 설정해줘야 한다.)
                    -3. url    : 사이트의 주소받는 입력창으로 (텍스트랑 비슷한데 http://문구가 있는지 체크해줌)
                    -4. email  : 이메일을 입력 받는 창(텍스트랑 비슷한데 뒤에 @가 있는지 파악)

                    .. 텍스트랑 유사한데 기능이 조금씩 주가됨
            
5. 기타 입력 창 태그

                    1. select   : combo box 형식으로 선택하는 것 (검색할 때 카테고리)
                                > 단독으로는 안쓰이고 자식태그로 옵션 태그를 가지고 있다. >옵션태크의 value값을 밑으로..( list )
                    2. datalist : input태그 text타입의 자동완성기능을 할 수 있게 하는 태그는
                                > 자식태그로 옵션 태그를 가지고 있다.
                    3. textarea : 여러 문장을 입력받게 하는 태그
                                > 긴 글을 쓰거나 게시판의 내용/ 소개글을 쓸 수 있도록
                                > 특정 크기를 속성으로 정해줘야함 (cols/ rows로)

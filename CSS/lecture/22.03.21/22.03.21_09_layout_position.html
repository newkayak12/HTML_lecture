position 속성


            태그를 원하는 위치에 배치하는 속성 값
            position 속성은 위치를 지정하는 top, left, right, bottom 속성과 함께 사용함
            position : static, relative, absolute, fixed

            relative : 부모의 position이 static일 떄  마지막 static elements의 위치를 0,0으로 사용
            absolute : 부모의 position이 static일 때 페이지의 0,0을 기준으로 삼는다.
            fixed : 특정태그를 지정 위치에 고정해서 출력

            부모가 relative/absolute이면 부모의 0,0이 시작지점이다.

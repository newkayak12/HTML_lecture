            padding과 margin

            padding : 테두리와 content 영역 사이즈 /elements내부에서 border와 content 사이의 간격을 조정하는 속성
                    > 선택자 { padding [-위치] 숫자(상대)/단위(절대)}
                    > 우리가 width/ height 주는건 content영역이다. 그래서 padding을 더 주면 그것보다 늘어날 수 있다.

            margin : 테두리 바깥과 다른 태그의 테두리 혹은 페이지 간의 간격  / elements와 elements, elements와 페이지 사이의 간격을 조정하는 속성
                    > 선택자 { margin [-위치]  숫자(상대)/단위(절대)}

background-repeat

            기본적으로 배경 이미지를 설정하면 바둑판으로 출력이되는데
            조장할 수 있는 속성으로는 (repeat/ repeat-x/ repeat-y/ no-repeat)

background-size

            배경이미지의 크기를 조정할 수 있는 속성
            기본으로 auto/ container : 이미지가 영역 안으로 모두 들어가도록 (이미지가 절대 짤리면 안 됨) / cover : 이미지가 컨테이너를 다 덮을 수 있도록(일부 잘려도 상관 없음)
                    숫자/백분률 : 크기 지정

contain

cover

background-position

             배경 이미지 위치를 조정하는 속성
             background-position : left||right||top||bottom||center||숫자(절대, 상대(%))
         
            background-position-x: right;
            background-position-y: top;
            이렇게 x,y축 나뉘어서 따로 조절하면 됨

            혹은 

            background-position : 위치 위치; 
            이걸로 한 번에도 가능

background-origin

             배경 이미지가 차지하는 범위를 설정하는 속성
             background-origin-borderbox, background-origin-paddingbox, background-origin-contentbox 가 있다.
             기본이 padding

background-attachment

             페이지가 스크롤에 의해 움직일 때 배경 이미지의 위치를 설정하는 속성
             scroll : 페이지가 움직이면 따라서 움직이는 속성 >> 페이지가 움직이면 안보이게 됨 
             fixed  : 페이지가 움직여도 설정한 위치에 있는 속성 >> 페이지가 움직여도 계속 보임

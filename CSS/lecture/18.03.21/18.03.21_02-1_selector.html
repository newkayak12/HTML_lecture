속성 선택자 : 태그 선택자(생략 가능) [속성명 비교연산자 속성값]

        태그에 설정되엉있는 속성과 속성값을 기준으로 선택하는 선택자
        input, img, a등의 속성을 기준으로 디자인을 적용할 때
        비교 연산
            = : 동등한 것 (완전 일치)
           ~= : 띄어쓰기를 기준으로 값을 확인
           |= : -를 기준으로 값을 확인
           ^= : 맨 앞글자를 기준으로 확인
           &= : 맨 뒷글자를 기준으로 확인
           *= : 포함 여부를 확인 

        eg_ '속성명 = 속성 값' 이런식으로 사용


^= 연산자 이용

^= : www.naver.com
^= : www-naver.com
^= : _wwwnaver.com

~= 연산자 이용

~= : naver com
~= : naver net
~= : co navar kr
~= : co kr naver

|=연산자 이용

|= : naver-net
|= : naver net
|= : naver~net
|= : www naver-net

*=연산자 이용

*= : www.google.com
*= : www.naver.com

동등 선택자 : +/~(전 선택자 + 후 선택자), (전 선택자 ~ 후 선택자)

        같은 위치(level)에 있는 태그를 선택하는 것
        
        + : 같은 위치 태그 중 앞에 오는 전 선택자의 다음에 오는 한 개의 후 선택자 태그

        ~ : 같은 위치 태그 중 앞에 오는 전 선택자의 다음에 오는 모든 후 선택자 태그

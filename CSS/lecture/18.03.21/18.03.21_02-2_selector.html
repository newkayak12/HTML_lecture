반응 선택자 : 선택자::active , 선택자::hover

        마우스의 이벤트에 의해 style이 적용되는 선택자
        1. active : 마우스로 클릭했을 때 style적용
        2. hover  : 마우스가 elements 태그 영역에 들어갔을 때 style 적용

클릭하면 변해라

문자 선택자 

        
        ::first-letter : 첫 번째 글자만 style 적용하기
        ::first-line   : 첫 번째 한 줄만 style 적용하기
        ::after        : 태그의 끝 문장에 문자 추가(내용 추가)
        ::before       : 태그의 시작에 문자 추가(내용추가)
        ::selection    : 마우스 드래그로 선택했을 때 디자인을 적용한다.

        계층을 기준으로 선택한다.

        선택자:first-child : 구조상 첫 번째 자식인 선택자 태그
        선택자:last-child  : 구조상 마지막 자식인 선택자 태그
        선택자:nth-child   : 구조상 n 번째 자식인 선택자 태그에서
